# RandomUtils

ReandomUtils is a C# port of [org.apache.commons.lang.math.RandomUtils](https://commons.apache.org/proper/commons-lang/javadocs/api-2.4/index.html).

## Support

| Type   | IsIncluded |
| ------ |:----------:|
| Bool   |     x      |
| Int    |     x      |
| Long   |     X      |
| Float  |     X      |
| Double |     X      |
